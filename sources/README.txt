Description:
This project focuses on latency bounded MAC scheduling in wireless networks. The system is implemented and analyzed using the NS-3 simulator with AODV routing protocol.

The performance is evaluated under different network conditions by varying the number of nodes and packet loads. Key metrics such as Packet Delivery Ratio (PDR), End-to-End Delay, Jitter, Throughput, Energy Consumption, and Communication Overhead are analyzed.

The results show that as node density and traffic load increase, delay and overhead also increase, while packet delivery ratio decreases due to congestion. This highlights the importance of efficient MAC scheduling techniques to maintain low latency and reliable communication.

Tools Used:
- NS-3 Simulator
- NetAnim
- Python (Matplotlib for graph generation)
